# Base16 Dark Ashes (Kelly) for Rider

This folder contains a JetBrains Rider theme plugin that matches the VS Code theme in this repository:

- IDE UI theme: `themes/base16-dark-ashes-kelly-rider.theme.json`
- Editor color scheme: `themes/base16-dark-ashes-kelly-rider.xml`

## Build plugin ZIP

```bash
cd rider-theme
./scripts/build-plugin.sh
```

Build output:

- `dist/base16-dark-ashes-kelly-rider-theme.zip`

## Install in Rider

1. Open `Settings` -> `Plugins`.
2. Click the gear icon -> `Install Plugin from Disk...`.
3. Select `dist/base16-dark-ashes-kelly-rider-theme.zip`.
4. Restart Rider.
5. Go to `Settings` -> `Appearance & Behavior` -> `Appearance`.
6. Select `Base16 Dark Ashes (Kelly)`.
